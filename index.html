About David Corbin

My name is David Corbin. I'm a full time student and part time software developer from Pittsburgh, Pennsylvania.

I develop and deploy software on Mac, Windows, and Linux. I've contributed to several popular, open source projects and built applications using a number of public API's including the Twitter API and the Github API.

When not building software, I enjoy playing trumpet and listening to a wide variety of music. I build electrical systems using Arduino and Raspberry Pi as well as discrete electronics. I also have a digital art project.
